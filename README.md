### 🌟 Filosofia de Desenvolvimento
Acredito no conceito de **"Adaptable Professional"**: o aplicativo não deve apenas funcionar, ele deve se adaptar ao ambiente do usuário.

* 🎨 **Pixel-Perfect:** Respeito rigoroso aos guias de estilo, paletas de cores e tipografia.
* ⚡ **Performance First:** Interfaces complexas construídas com hierarquias de layout simplificadas para garantir 60fps (ou mais) no scroll e nas transições.
* 🧹 **Código Limpo:** A UI não deve misturar lógica de negócio. *State Hoisting* e injeção de dependência são meus melhores amigos.
